Everyone has a setup post. Here's mine, kept short.

## Editor

**Xcode** for iOS work. There's no alternative and it's gotten good enough that I don't resent it anymore. **Android Studio** for Kotlin. **Cursor** for everything else — Python, .NET, web, config files.

I tried to use a single editor for everything. It doesn't work. Each platform's tooling is too tightly integrated with its IDE.

## Terminal

Ghostty with zsh. Nothing exotic. I have aliases for `git status`, `git log --oneline`, and a few project-specific shortcuts. That's it.

## Git workflow

Feature branches off `develop`, PRs to `main` via review. I use `git rebase` to keep history clean before merging. I almost never use a GUI for git — the CLI is faster once you know the ten commands you use daily.

## API testing

Bruno. Replaced Postman a while ago and haven't missed it. Collections live in the repo, git-tracked, no account required.

## Design handoff

Figma. The iOS and Android plugins are good enough that I rarely need to measure anything by hand.

## Note-taking

Obsidian for technical notes and anything I want to find later. Apple Notes for everything else. I've tried to consolidate and always regret it.

## What I deliberately don't have

- A complicated dotfiles setup
- Multiple terminal multiplexers
- "Productivity systems" with more than three layers

Every tool I add has to earn its place by making something faster or reducing a decision. Most things don't clear that bar.
